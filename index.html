Setup Required

To access your playlists, you need a free Spotify Developer Client ID. It takes about 2 minutes to get one.

How to get your Client ID
  1. Go to developer.spotify.com/dashboard and log in
  2. Click Create App
  3. Fill in any name/description (e.g. "Playlist Picker")
  4. For Redirect URI, enter exactly:  http://localhost  — wait, actually copy the URL of this page and paste it as the redirect URI
  5. Check the Web API checkbox and save
  6. Copy the Client ID from the app dashboard and paste it below

Your credentials are never sent anywhere — all auth happens directly between your browser and Spotify.
